# color-series-10

生成一个系列10种颜色，可用于主题色搭配选择。

示例1
```javascript
let colorPalette = require('color-series-10/dist/toggle-rgb-hsv.min').colorPalette;
// 或
// let colorPalette = require('color-series-10/dist/toggle-rgb-hsv.min').colorPalette;
console.log(
    ...colorPalette('#ff0000')
);
```

结果
```bash
#ffccd5 #ffa3af #ff7a88 #ff525d #ff2930 #ff0000 #d90700 #b30c00 #8c0e00 #660e00
```
